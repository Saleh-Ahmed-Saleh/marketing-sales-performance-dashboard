# 📊 Sales & Marketing Analytics Dashboard (Power BI)

## 📁 Project Overview
This project presents a **professional Sales & Marketing Analytics Dashboard** built using Power BI and real-world business data.

- Data Cleaning  
- Data Modeling  
- KPI Development using DAX

The solution enables stakeholders to **monitor performance, analyze trends, and optimize marketing strategies** using data.

## 🧱 Data Model

The project follows a **Star Schema**:

### Fact Table:
- Sales / Marketing Fact Table (Revenue, Cost, Sessions, Views)

### Dimension Tables:
- `dimCountry`
- `dimCustomer`
- `dimpRroduct`
- `dimCustomerPlan`
- `dimLeadSource`
- `dimpMarketingCampaign`
- Calendar Table

### Relationships:
- One-to-Many relationships from dimensions to fact table  
- Calendar table used for time intelligence (YoY, MoM analysis)

---

## ⚙️ Data Preparation & Modeling
- Cleaned and transformed raw data using **Power Query**
- Created a **Calendar table** for time-based analysis
- Built relationships across multiple tables
- Optimized model performance

## 📊 Key Measures (DAX)

### 💰 Revenue Metrics
- Revenue  
- Revenue PY (Previous Year)  
- Revenue PM (Previous Month)  
- Revenue Variance  
- YOY Revenue Growth  

### 💸 Cost Metrics
- Cost  
- Cost PY  
- Cost Variance  
- YOY Cost Growth Rate    

### 📈 Profit Metrics
- Profit  
- Profit PY  
- Profit Variance  
- YOY Profit Growth  

### 👁️ Engagement Metrics
- Views  
- Views PY  
- YOY Views Growth Rate  

### 🌐 Conversion Metrics
- Avg Conversion Rate  
- Avg Conversion Rate PY  
- Avg Conversion Rate Variance  
- YOY Avg Conversion Rate   

### 📊 Additional Metrics
- Sessions  
- Invoices  
- Scoring  
- Avg Rating / Avg Score  
- Max Revenue
